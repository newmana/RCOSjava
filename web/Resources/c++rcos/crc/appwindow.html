RCOS C++ CRC for class AppWindow

Name:

 AppWindow

Comment:

 This is a "Window". Instantiating one of these gives a work area (the graphic window) with anchors for all three button types, and some options, like menu bars and close gadget. The window can be system modal (no menu bar, all input limited to the window), or modeless (menu bar, inputs go to it and its modeless ancestors). NOTE that (mostly) there will only ever be one instance of the "Window" class and that will belong to the one instance of the "Application" class. Child windows are linked to the original window instance by calls to , so all the constructor need do is invoke that member to do create the head element. In fact, this whole class probably should be private to the "App" class.

MemoryManager CRC

Name:

 MemoryManager

Comment:

 Responsible for allocating memory to processes and operating system should also be used to read and write memory locations. Based on MessageHandler class.

Collaborators:

 MessageHandler
MemoryBlock - a block of memory allocated by the MemoryManager to a process

Responsibilities:

  • HandleMessage

  • MemoryBlock theBlock = AllocateMemory( int size );
    Given a size of memory requested find and allocate memory block

  • DeAllocateMemory( MemoryBlock theBlock )

  • CompactMemory()
    Optional function might be used on early systems

  • MemoryBlock theBlock = ReadMemory( int start, int size )
    return a memory block in response for a read memory request

  • int status = WriteMemory( int start, int size, MemoryBlock theBlock )
    format may change

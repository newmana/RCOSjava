Kernel CRC

Name:

 Software, Kernel

Comment:

 A micro kernel based system. It handles system calls and returns values from other operating system components and process switching .

Collaborators:

 Interrupt - store information about type, source etc
Interrupt - store information about type, source etc

Responsibilities:

  • blockCurrentProcess - when a process is blocking.
  • killProcess - when a process has been killed by the user or external process.
  • nullProcess - when no process is currently running.
  • setQuantum - set the number of ticks before a process is pre-empted.
  • switchProcess - take the current process and replace it with the given process.
  • pause/unpause - pause and unpause the CPU.
  • performInstructionCycle - call schedule on process scheduler and perform an instruction on the CPU.
  • setProcessCode, setProcessStack - set the currently executing process process code and process stack.
  • setContext - set the currently executing process context.
  • runningProcess - true if a process is currently executing.
  • getCurrentProcess - returns the currently executing process.

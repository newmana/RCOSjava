Disk

Class:

 Hardware, Disk

Name:

 Disk

Comment:

 Allows the disk scheduler to similate access times and other characteristics of a a physical disk drive. Also allows reading and writing of disk sectors. You can perform the follow simulated operations:
  1. Successive sector read,
  2. Successive track read,
  3. Sequential access of an arbitary length file, and
  4. Random access of an arbitary length file.
Alternatively, your own calculations can be applied by directly accessing the disk characteristics.

Collaborators:

 Disk Scheduler - Accesses this directly to read/write and to simulate real reading and writing.

Responsibilities:

  • Constructor - create a disk with track number and size, sector size, latency, start-up time and rotational speed.
  • Calculate Sectors to Read - given the number of bytes based on the number of sectors.
  • Calculate Tracks to Read - given the number of bytes base on the number of tracks.
  • Calculate Seek Time - given the number of tracks return the seek time.
  • Calculate Transfer Time - given the number of bytes return the time take to get it off the disk.
  • Calculate Successive Track and Sector Read.
  • Calculate Sequential and Random access.
  • Read and write data.

/ Resources / Design / Messages / IPCManagerAnimator


As this Animator doesn't allow the user to modify the memory manager, shared memory or semaphores directly, no messages are sent. The messages that are received are used to display the current semaphore, shared memory and memory usage.

Currently, reading/writing bytes/pages are treated the same.

 None

Processed by IPCManagerAnimator

 ReadBytes, WriteBytes, ReadPage, WritePage

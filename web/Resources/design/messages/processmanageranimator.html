The process manager is designed to allow users to basically interact with the P-Code CPU and processes in general. The Step and Run buttons effect the CPU and the Kill button effects the processes.

It receives messages so that only existing processes can be killed.

Sent by ProcessManagerAnimator

To Name
ProgramManager
 Run, Step, Kill

Processed by ProcessManagerAnimator
